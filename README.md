# Vue 2 + Vuetify 1.5 Mock Repo (test fixture)

A deliberately outdated app for testing the Codebase Migration Assistant. Every file
plants specific Vue 2 → Vue 3 / Vuetify 1.5 → 3 migration issues (global filters,
`.sync`, event bus, `$children`, old slots, functional components, Vuetify grid /
`v-btn flat` / `v-list-tile`, outdated deps). For *analysis*, not for running as-is.
